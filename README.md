# Minecraft

## 조작
### 로비
- [   ESC   ] : 종료
- [  SPACE  ] : 게임 시작

### 게임
- [   ESC   ] : 로비화면으로
- [   TAB   ] : 시점 변경
- [   F3    ] : 로그 출력 토글
- [ W/A/S/D ] : 이동
- [  SPACE  ] : 점프
- [   +/=   ] : 이동속도 빠르게/원래대로
- [ 1/2/3/4 ] : 설치할 블럭 선택(벽돌/조약돌/철블럭/횃불)
- [   I/i   ] : 초기화
- [   F/f   ] : 카메라 고정 토글
- [  [ / ]  ] : 렌더거리 조절
